→ Student Expense Tracker

A web-based expense tracking application designed for students to manage daily and monthly spending efficiently.

→ Features
- Multi-user login system (frontend-based)
- Category-wise expense tracking (Food, Travel, Clothing, Petrol, etc.)
- Daily spending limit with real-time remaining balance
- Monthly budget enforcement
- Expense search and filtering
- Persistent per-user data storage using browser localStorage
- Clean and responsive user interface

→ Tech Stack
- HTML
- CSS
- JavaScript

→ How to Run Locally
1. Download or clone the repository
2. Open `login.html` in any modern web browser
3. Login using any User ID and Password
4. Start tracking expenses

→ Project Highlights
- Implements strict daily and monthly budget enforcement to prevent overspending
- Uses client-side logic to simulate multi-user behavior
- Designed with scalability in mind for future backend integration
